## 커리큘럼 학습 단계 (curriculum.py 실제 기준)

총 **15단계** (4 기초 + 10 Two-Circle Headon + 1 Final)

```
Stage 0   flight_survival        고정 타겟 / 60s   → 생존·스로틀 제어
Stage 1   target_pursuit         고정 타겟 / 120s  → 기수 정렬·접근
Stage 2   wez_approach           로이터 타겟 / 180s → WEZ 진입
Stage 3   autopilot_pursuit      오토파일럿 타겟 / 240s → 이동 타겟 추격
Stage 4   two_circle_headon_a000  alpha=0°
Stage 5   two_circle_headon_a020  alpha=20°
  ...     ...                    alpha 20° 씩 증가
Stage 13  two_circle_headon_a180  alpha=180°
Stage 14  full_dogfight          BT 상대 / 300s   → 완전 교전
```

> ==Stage 0·1 상대는 Fixed(고정 표적)이며 BT가 아님. Stage 0은 damage_scale = 0 으로 데미지 보상 없음.==

### 단계 진급 조건 (advance_conditions)

| Stage | 조건 | 임계값 |
|-------|------|--------|
| 0 flight_survival | crash_rate < | 20% |
| 1 target_pursuit | ep_min_distance < 2000m AND crash_rate < | 15% |
| 2 wez_approach | ==win_rate ≥ 10% **OR** ep_wez_steps ≥ 10 스텝 (둘 중 하나)==  | 유일한 OR 조건 |
| 3 autopilot_pursuit | ep_min_distance < 1500m AND ep_wez_steps ≥ 5 AND crash_rate < | 20% |
| 4–13 two_circle_headon | win_rate ≥ 70% AND crash_rate < | 30% |
| 14 full_dogfight | 없음 (최종 단계) | — |

## 단계별 보상 오버라이드

| Stage | survival | pursuit_scale | damage_scale | win_reward |
|-------|----------|---------------|--------------|------------|
| 0 | **+0.05** | 0.0 | 0.0 | 0.0 |
| 1 | +0.01 | 0.5 (angle 60°) | 0.0 | 20.0 |
| 2 | 0.0 | 0.3 | 20.0 | 100.0 |

| 4–14 | 기본값 사용 (오버라이드 없음) | — | — | — |

## 핵심 하이퍼파라미터 (my_train.py 기준)

```python
TRAINING_CONFIG = {
    "algorithm": "sac",
    "iterations": 50,
    "reward_module": "student.my_reward",
}

ENV_CONFIG = {
    "observation_mode": "tactical16",
    "target_mode": "behavior_tree",
    "max_engage_time": 300.0,
    "episode_step_limit": 18000,
}

RL_CONFIG = {
    "framework": "torch",
    "num_env_runners": 1,
    "num_envs_per_env_runner": 1,
    "lr": 3e-4,          # actor_lr / critic_lr / alpha_lr 공통
    "gamma": 0.99,
    "train_batch_size": 4096,
    "minibatch_size": 256,
    "tau": 0.005,         # SAC 소프트 업데이트
    "target_entropy": "auto",
}
```

## 훈련 실행 방법

```bash
# 1. 기본 SAC 훈련
python student/my_train.py

# 2. 커리큘럼 훈련
python train_curriculum.py --stages-module student.my_curriculum

# 3. 직접 실행 (train_rllib.py)
python train_rllib.py \
    --algorithm sac \
    --iterations 100 \
    --reward-module student.my_reward \
    --observation-mode tactical16 \
    --output-name team01 \
    --output-tag v1
```

## SAC-LSTM (선택 옵션)

RLLibLstm 패치 적용 후 사용 가능:
```bash
python train_rllib.py --use-lstm-sac --lstm-cell-size 64 --max-seq-len 8
```
