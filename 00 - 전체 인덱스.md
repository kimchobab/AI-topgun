## 🏗️ 시스템 핵심 컴포넌트

```
DogFightEnvWrapper
├── DogFightEnv (gymnasium.Env)
│   ├── FighterSim.JSBSim (ownship)        ← Blue / 아군
│   ├── FighterSim.JSBSim (target)         ← Red / 적기
│   ├── GeometryInfo                        ← ATA/AA/HCA/LOS 계산
│   ├── ActionProvider (ownship/target)     ← BT / RL / Hybrid
│   └── JSBSimWrapper (battle space)
└── 학생 커스터마이징 포인트
    ├── reward_fn         ← 보상 함수 재정의
    ├── observation_fn    ← 관측 함수 재정의
    └── bt_rule_xml       ← BT 규칙 XML 수정
```
