# 🚨 종료 조건

[[00 - 전체 인덱스|← 인덱스로]]

---

## terminated vs truncated

| 구분 | 의미 | 보상 처리 |
|------|------|----------|
| `terminated=True` | 물리적으로 에피소드 종료 (사망/충돌/승패) | 터미널 보상 적용 |
| `truncated=True` | 시간 제한으로 강제 종료 | 터미널 보상 미적용 |

---

## 종료 조건 우선순위 (termination.py)

```python
# 순서대로 평가 — 먼저 매칭되는 조건으로 종료
1. FDM Update Fail         → terminated (물리 연산 오류)
2. ownship altitude below min → terminated (아군 최저 고도 이하)
3. target altitude below min  → terminated (적기 최저 고도 이하)
4. ownship destroyed       → terminated (아군 HEALTH ≤ 0)
5. target destroyed        → terminated (적기 HEALTH ≤ 0)
6. fuel fail               → terminated (연료 소진)
7. two circle headon guard fail → terminated (Two-Circle 가드 위반)
8. max time out            → truncated  (SIM_TIME > 300초)
9. episode step limit      → truncated  (스텝 수 ≥ 18000)
```

---

## 종료 조건 → 에피소드 결과(outcome) 매핑

| end_condition | terminated | truncated | ownship_health | target_health | outcome |
|---|:---:|:---:|---|---|---|
| target destroyed | ✓ | — | > 0 | ≤ 0 | **win** |
| ownship destroyed | ✓ | — | ≤ 0 | > 0 | **loss** |
| ownship altitude below min | ✓ | — | — | — | **crash** |
| FDM Update Fail | ✓ | — | — | — | **crash** |
| two circle headon guard fail | ✓ | — | — | — | **loss** |
| 기타 terminated | ✓ | — | ≤ 0 또는 같음 | — | **draw** |
| max time out | — | ✓ | — | — | **timeout** |
| episode step limit | — | ✓ | — | — | **timeout** |

> `truncated=True`인 경우(max time out / episode step limit) 터미널 보상은 적용되지 않는다. `terminated=True`일 때만 `win_reward`, `loss_penalty` 등이 발동됨.

## Two-Circle Headon Guard

헤드온 커리큘럼에서 올바른 선회 방향을 강제:

```python
alpha_deg = 교전 초기 헤딩 오프셋 각도

if alpha_deg <= 80°:
    # 초반: ATA가 90° 초과하면 실패 (잘못된 방향 선회)
    guard_fail = (current_ata > 90°)

elif alpha_deg <= 140°:
    # 중반: ATA가 alpha + margin 초과하면 실패
    guard_fail = (current_ata > alpha + 10°)

else:
    # 고각도: 가드 비활성
    guard_fail = False
```

## 주요 파라미터

| 파라미터 | 기본값 | 설명 |
|----------|--------|------|
| `max_engage_time` | 300.0 s | 최대 교전 시간 |
| `episode_step_limit` | 18000 | 최대 RL 스텝 수 |
| `min_altitude` | 300.0 m | 최소 안전 고도 |
| `geometry_guard.enabled` | False | Two-Circle 가드 활성화 |
| `geometry_guard.crossed_ata_limit_deg` | 90.0 | 초반 ATA 한계 |
| `geometry_guard.turn_margin_deg` | 10.0 | 중반 ATA 여유 |
