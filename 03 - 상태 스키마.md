## StateIndex enum (state_schema.py)

항공기 상태 벡터는 최소 **46개** 필드의 float32 배열입니다.

| 인덱스 | 이름 | 단위 | 설명 |
|--------|------|------|------|
| 0 | `N` | m | NED 북쪽 위치 |
| 1 | `E` | m | NED 동쪽 위치 |
| 2 | `D` | m | NED 아래쪽 위치 (음수 = 고도) |
| 3 | `ROLL` | deg | 롤 각도 [-180, 180] |
| 4 | `PITCH` | deg | 피치 각도 [-90, 90] |
| 5 | `YAW` | deg | 요 각도 (헤딩) [0, 360] |
| 12 | `KCAS` | knots | 교정 대기속도 |
| 23 | `FUEL` | — | 연료량 (0 = 소진) |
| 41 | `SIM_TIME` | sec | 시뮬레이션 경과 시간 |
| 42 | `LAT` | deg | 위도 |
| 43 | `LON` | deg | 경도 |
| 44 | `ALT` | m | 고도 (MSL) |
| 45 | `HEALTH` | [0,1] | 체력 (0 = 격추) |

> **주의**: 인덱스 6–11, 13–22, 24–40은 JSBSim 내부 FDM 상태 변수 (속도 컴포넌트, 각속도, 가속도 등)

## 위치 표현

```python
# NED 좌표계 (North-East-Down)
# D < 0 이면 공중에 있음 (예: D=-7000 → 고도 7000m)

# LAT/LON/ALT로 변환 (pymap3d 사용)
lla = pm.ned2geodetic(N, E, D, origin_lat, origin_lon, origin_alt)
```

## 체력(HEALTH) 감소 메커니즘

```
초기 HEALTH = 1.0
매 스텝마다 WEZ 내에 있으면:
  damage = ((max_range - distance) / base_range) * delta_t
  HEALTH -= damage
HEALTH <= 0 → 격추 (terminated)
```

## 주요 헬퍼 함수

```python
from dogfight.sim.state_schema import StateIndex, position_ned, attitude_rpy

# 위치 벡터 추출 (N, E, D)
pos = position_ned(state)   # state[0:3]

# 자세 벡터 추출 (roll, pitch, yaw)
att = attitude_rpy(state)   # state[3:6]
```
