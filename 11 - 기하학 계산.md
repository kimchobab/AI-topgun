## 핵심 각도 4종

| 약어 | 이름 | 설명 | 범위 |
|------|------|------|------|
| **ATA** | Antenna Train Angle | **아군 기수**에서 적기까지의 각도 | [-180°, 180°] |
| **AA** | Aspect Angle | **적기 꼬리**에서 아군까지의 각도 | [-180°, 180°] |
| **HCA** | Heading Cross Angle | 아군/적기 **헤딩 벡터** 사이각 | [-180°, 180°] |
| **LOS az/el** | Line of Sight | 아군 기준 적기 방위각/고도각 | az[-180,180], el[-90,90] |

---

## 전술적 의미

```
   AA=0° (적기 정면)      AA=180° (적기 후방)
   ← [적기 기수] →→→→→ [적기 꼬리] →
     (헤드온, 위험)        (아군이 뒤를 잡음 = 유리)

아군 →→
[기수]
     ↘ ATA
        [적기]
   (ATA 작을수록 정면 조준)
```

> **AA 정의 기준 (코드)**: 적기 기수 방향(+X)과 아군 방향 벡터 사이각.
> AA=0° = 아군이 적기 정면 (헤드온), AA=180° = 아군이 적기 후방 (유리).

| 각도 | 값 | 전술 상황 |
|------|-----|----------|
| ATA = 0° | 정면 조준 | 최적 사격 위치 |
| ATA < 1° + 거리 조건 | WEZ 진입 | 데미지 발생! |
| AA = 0° (12시) | 적기 정면 대향 | 헤드온 위험 |
| AA = 180° (6시) | 적기 바로 뒤 | 유리한 위치 |
| HCA = 0° | 같은 방향 비행 | 추격 상황 |

## ATA 계산 (Antenna Train Angle)

```python
def _get_antenna_train_angle(ownship_ned, target_ned, proj=False):
    # ownship의 Body Frame으로 target 방향 변환
    phi, theta, psi = ownship_ned[3:6] * D2R  # roll, pitch, yaw

    # 회전 행렬 (NED→Body)
    Tx = rotation_x(phi)
    Ty = rotation_y(theta)
    Tz = rotation_z(psi)
    T_nb = Tx @ Ty @ Tz

    # 타겟 방향 단위 벡터 (NED)
    p_ned = (target_ned[:3] - ownship_ned[:3]) / dist

    # Body Frame으로 변환
    p_body = T_nb @ p_ned

    # 3D: 아크코사인으로 각도 계산
    angle = arccos(p_body[0]) * R2D
    # 부호: p_body[1] 방향으로 좌우 구분
```

## LOS 각도 (Line of Sight)

```python
def _get_los_angle(ownship_ned, target_ned):
    # 아군 Body Frame에서 타겟의 방위각(az)과 고도각(el)
    dis_body = T_nb @ dis_unit_ned
    az = arctan2(dis_body[1], dis_body[0]) * R2D  # -180 ~ 180
    el = -arcsin(dis_body[2]) * R2D               # -90 ~ 90
```

## 회전 행렬 (NED → Body)

$$T_{NED \rightarrow Body} = T_x(\phi) \cdot T_y(\theta) \cdot T_z(\psi)$$

```python
# 오일러 각도 기반 (ZYX 순서, NED 좌표계)
Tx = [[1, 0,    0   ],
      [0, cos φ, sin φ],
      [0,-sin φ, cos φ]]

Ty = [[cos θ, 0,-sin θ],
      [0,     1, 0    ],
      [sin θ, 0, cos θ]]

Tz = [[cos ψ, sin ψ, 0],
      [-sin ψ,cos ψ, 0],
      [0,     0,     1]]
```

## proj=True vs proj=False

| `proj` | 계산 방식 | 용도 |
|--------|----------|------|
| `False` | 3D (아크코사인) | 정확한 3D 공간 각도 |
| `True` | 2D 수평 투영 (아크탄젠트) | 평면 기동 분석, 부호 명확 |

> WEZ 판단은 `proj=False` 3D ATA 사용
> tactical16 obs[9]도 3D ATA
