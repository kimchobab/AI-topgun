## 생성자 주요 파라미터

| 파라미터 | 타입 | 기본값 | 설명 |
|----------|------|--------|------|
| `env_config` | dict | None | 환경 전체 설정 (config.py DEFAULT_ENV_CONFIG) |
| `reward_fn` | callable | None | 학생 정의 보상 함수; None이면 reward.py 기본값 |
| `observation_fn` | callable | None | 커스텀 관측 함수 |
| `observation_size` | int | None | 커스텀 관측 크기 |
| `ownship_action_provider` | ActionProvider | None | Blue 항공기 제어기 |
| `target_action_provider` | ActionProvider | None | Red 항공기 제어기 |

## 핵심 설정값 (DEFAULT_ENV_CONFIG)

```python
sim_hz           = 60       # 물리 시뮬레이션 주파수 (Hz)
max_engage_time  = 300.0    # 최대 교전 시간 (초)
episode_step_limit = 18000  # 최대 스텝 수
min_altitude     = 300.0    # 최소 고도 (m) — 이하 시 추락 판정
observation_mode = "classic12"   # 코드 기본값. 실습/훈련 권장값은 "tactical16"
ownship_control_mode = "rl"      # rl / behavior_tree / fixed / loiter
target_mode      = "behavior_tree"
```

> **observation_mode 기본값 vs 권장값**
> - 코드 기본값: `classic12` (아무 설정 없이 실행하면 이 값이 선택됨)
> - 실습/훈련 권장값: `tactical16` (완전 정규화, 전술 피처 포함)
> - `my_train.py`, `run_local_dogfight.py`, `my_submission.py` 세 곳 모두 동일한 값으로 명시적으로 설정할 것

## 행동 공간 / 관측 공간

```python
# 행동 공간: 4D 연속 [-1, 1]
action_space = Box(low=-1, high=1, shape=(4,), dtype=float32)
# 내부 변환: throttle = (throttle_rl + 1) / 2  → [0, 1]

# 관측 공간
# tactical16: Box(-1, 1, (16,))
# relative14: Box(-inf, inf, (14,))
# classic12 : Box(-inf, inf, (12,))
```

## 핵심 메서드

| 메서드 | 역할 |
|--------|------|
| `reset()` | 에피소드 초기화, 시나리오 적용, 관측 반환 |
| `step(action)` | 물리 스텝 실행, 보상·종료 계산 |
| `update_damage()` | WEZ 기반 데미지 적용 |
| `get_observation()` | 현재 관측 벡터 반환 |
| `change_init_position()` | 초기 위치 변경 |
| `add_random_init_position()` | 랜덤 초기 위치 추가 |
| `make_tacviewLog()` | CSV 비행 로그 저장 |

## step() 반환값

```python
obs, reward, terminated, truncated, info = env.step(action)

# info 딕셔너리 주요 키
{
  "end_condition": str,         # 종료 이유 문자열
  "outcome": str,               # "win" / "loss" / "draw" / "crash" / "timeout"
  "ownship_health": float,      # 아군 체력 [0,1]
  "target_health": float,       # 적기 체력 [0,1]
  "reward_components": dict,    # 스텝별 보상 컴포넌트
  "ep_wez_steps": int,          # 에피소드 WEZ 내 스텝 수
  "ep_min_distance": float,     # 에피소드 최소 교전 거리
  "final_ata_deg": float,       # 종료 시 ATA 각도
}
```
