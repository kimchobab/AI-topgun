# 🛠️ 수동 실행 가이드 (Quick Reference)

> 모든 명령은 **Release 폴더**에서 실행

```powershell
Set-Location "C:\Users\user\workspace\topgun\AI Pilot Top Gun Challenge 2026\AIP_LIB\DogFightEnv\Release"
```

---

## 1. 학습 실행

### 기본 SAC 학습 (권장)
```powershell
python train_rllib.py `
  --algorithm sac `
  --iterations 50 `
  --output-name team01 `
  --output-tag sac_v2 `
  --reward-module student.my_reward `
  --observation-mode tactical16 `
  --target-mode behavior_tree `
  --target-behavior-dll AIP_BASE_target.dll
```

### my_train.py wrapper (간단 버전)
```powershell
python student\my_train.py --iterations 50
```

### 커리큘럼 학습
```powershell
python train_curriculum.py `
  --algorithm sac `
  --reward-module student.my_reward `
  --output-name team01 `
  --output-tag curriculum_v1
```

---

## 2. 로컬 교전 테스트 (RL vs BT)

```powershell
python run_local_dogfight.py `
  --ownship-backend rl `
  --ownship-bundle-dir artifacts\models\team01\sac_v2 `
  --target-backend bt `
  --observation-mode tactical16 `
  --save-log
```

> 승률 50% 이상이면 제출 준비 완료

---

## 3. BattleServer 뷰어 연결

### Step 1 — 뷰어 실행
```powershell
Start-Process "C:\Users\user\workspace\topgun\AI Pilot Top Gun Challenge 2026\BattleServer_Low\DogFightViewer.exe"
```
→ 창에서 **Start 버튼 클릭** (UDP 9999 포트 오픈됨)

### Step 2 — RL 모델 연결
```powershell
python run_unreal_inference.py `
  --mode rl `
  --bundle-dir artifacts\models\team01\sac_v2 `
  --observation-mode tactical16 `
  --team-name team01 `
  --server-ip 127.0.0.1 `
  --server-port 9999 `
  --packet-monitor
```

### BT 모드 연결 (모델 없이 테스트)
```powershell
python run_unreal_inference.py `
  --mode bt `
  --bt-dll AIP_BASE.dll `
  --bt-rule-xml Rule_forTraining.xml `
  --team-name team01 `
  --server-ip 127.0.0.1 `
  --server-port 9999
```

## 5. 대회 서버 제출

`student\my_submission.py` 수정 후 실행:
```python
TEAM_NAME = "team01"
SERVER_IP  = "221.151.77.208"   # 대회 서버 IP
BUNDLE_DIR = "artifacts/models/team01/sac_v2"
OBSERVATION_MODE = "tactical16"
ACTION_REPEAT = 6
```
```powershell
python student\my_submission.py
```

## 6. 체크리스트 (제출 전)

- [ ] `OBSERVATION_MODE` — 학습 시 사용한 것과 동일?
- [ ] `ACTION_REPEAT = 6` — my_train.py step_ratio와 동일?
- [ ] `BUNDLE_DIR` — 실제 모델 경로 맞나?
- [ ] `Rule.xml` / `Rule_forTraining.xml` — Release 폴더에 있나?
- [ ] 로컬 승률 50% 이상 확인?
